Why SafeNavi

공공데이터와 설명 가능한 기준을 사용합니다

단순히 결과만 제시하지 않고, 어떤 데이터를 사용했는지와 추천점수가 어떤 요소로 구성됐는지를 함께 확인할 수 있도록 설계했습니다.

01

공공데이터 기반

행정안전부 긴급재난문자, 대피소·수용시설, 국민행동요령과 기상 데이터를 활용합니다.

02

지역 맞춤 분석

선택한 주소나 현재 위치와 관련된 재난정보를 우선 필터링해 지역 중심 결과를 제공합니다.

03

설명 가능한 추천

거리, 재난 적합도, 수용정보와 출처 점수를 분리해 추천 이유를 사용자가 확인할 수 있습니다.

04

공식 안내 우선

SafeNavi 결과는 참고정보이며, 실제 재난 시에는 정부·지자체·소방기관의 안내를 우선합니다.

Technology

데이터, 위치정보와 AI를 하나로 연결했습니다

재난문자 분류, 위험점수 계산, 위치 기반 대피소 추천, RAG 행동요령 검색과 가족 공유 기능을 하나의 흐름으로 제공합니다.

Flask PostgreSQL PostGIS 공공데이터 API 위치 기반 추천 RAG 행동요령 카카오 주소검색 OpenStreetMap

How It Works

SafeNavi 이용 흐름

1

위치 선택

주소를 검색하거나 현재 위치를 확인합니다.

2

지역 위험 분석

재난문자와 기상정보를 지역 기준으로 분석합니다.

3

대피소 추천

거리와 재난 적합도를 계산해 추천 순위를 제공합니다.

4

행동·공유

행동요령을 확인하고 가족에게 안전상태를 공유합니다.

서비스 이용 안내

SafeNavi의 안전점수와 추천 결과는 재난 상황 판단을 돕기 위한 참고정보입니다. 실제 위험 상황에서는 긴급재난문자, 현장 통제, 정부와 지자체의 공식 대피 안내를 따라주세요.
